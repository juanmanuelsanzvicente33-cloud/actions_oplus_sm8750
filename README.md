### ⚙️ 内核信息

➜ **芯片平台**: `sm8750 | 8 Elite | sun`

➜ **内核版本**: `6.6`

➜ **Android 版本**: `15 VanillaIceCream (can be used on later versions too!)`

➜ 可用于: `OxygenOS / ColorOS`

---

### 📝 
## 功能特性

➜ **ccache**（首次构建后，后续构建将快得多！）

➜ **mountify** 支持

➜ 基带守护（Baseband Guard）

➜ BetterNET 网络优化

➜ **KPM** 支持（`SukiSU Ultra & ReSukiSU`）

➜ **lz4** 与 **zstd** 补丁

➜ **ZRAM 配置模块**

➜ **ADIOS I/O 调度器** & **Re:Kernel**

➜ **自定义**内核后缀（默认模拟**原厂**标识）

➜ **自定义**内核构建时间（默认模拟**原厂**标识）

➜ Multimanager 支持（**仅 ReSukiSU & KernelSU Next！**）：  
`官方 KSU、5ec1cff KSU、rsuntk KSU、KowSU、MamboSU、KernelSU NEXT、WildKSU`

➜ Unicode 绕过（可使用 **LSPosed 模块**进行无痛修复）

### 🤖 编译器改动

➜ 使用 **Neutron Clang**

➜ 合并汇编器与编译器

➜ 启用编译器 **POLLY**

➜ 启用 **O2** 编译优化参数

---
### ✨ 如何编译！

➜ **Fork** 项目（包含所有分支！）

➜ Actions ➜ 启用工作流

➜ 选择你偏好的变体

➜ 选择你的 **语言**（**仅用于配置内核**）➜ 分支：**english** | **zh**

➜ 根据你的 **偏好** 进行配置

➜ 运行工作流
